# gitkai

Terminal cockpit for git repos, worktrees, and AI coding sessions.

One board for every repo on your machine: branches, worktrees, dirty state, and
the AI coding agents working in them — live, in a single TUI.

- No login. No telemetry. No daemon. No cloud. Free.

## Install

```sh
brew install ooookai/tap/gitkai
```

macOS (Apple Silicon + Intel) and Linux (arm64 + x64).
